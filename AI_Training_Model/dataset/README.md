## SAMPLE cho Train Model
Folder này chứa các Sample cho quá trình training bao gồm 3 folder
* Hen suyễn (288 mẫu)
* Các bệnh khác (280 mẫu)
* Rác, tạp âm (180 mẫu)
Ngoài rác và tạp âm ra, tất cả các mẫu được cung cấp bởi trang web Kaggle

Bạn cũng có thể download bộ Sample của Kaggle và sử dụng cho các mục đích khác theo đường link sau
[Asthma Detection Dataset Version 2 trên Kaggle](https://www.kaggle.com/datasets/mohammedtawfikmusaed/asthma-detection-dataset-version-2?resource=download)
